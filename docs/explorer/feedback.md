We welcome feedback, bug reports, and feature suggestions.

## Where to provide feedback

The best place to provide feedback is via GitHub:

https://github.com/jimchurches/myebirdstuff

You can:

- Report bugs  
- Request new features  
- Ask questions about behaviour  
- Suggest improvements  

---

## Using GitHub Issues

If you are familiar with GitHub:

- Open an issue in the repository  
- Provide as much detail as possible  
- Include screenshots or examples where helpful  

If you are not familiar with GitHub:

- Follow the link above  
- Select “Issues”  
- Click “New Issue”  
- Enter a description of your feedback  

---

## In-app link

A link to the GitHub repository is also available in the application sidebar.

---

## Notes

- Feedback is reviewed as time permits  
- Not all feature requests will be implemented  
- Clear, concise reports help prioritise and resolve issues more quickly  

---

## Summary

- Use GitHub Issues for all feedback  
- Include details where possible  
- The sidebar link in the app provides quick access
